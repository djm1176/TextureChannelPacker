Texture Channel Packer

The Physically Based Rendering pipeline of many game engines allow you to use multiple physically based greyscale textures. Since these materials are greyscale, rather than waste 3 RGB channels to convey intensity, we can combine 3 (and even a 4th into the alpha channel!) of these textures into a single RGBA image, saving on space.

At runtime, the game engine's shaders will decompose each channel into the original luminance values for each of these maps.

Use the tool below to upload up to 4 images to be combined into a single RGBA image. Currently, only the red channel from each image is mapped to one of the new 4 channels.

Unity's HDRP uses (R) Metallic, (G) Ambient Occlusion, (B) Detail, and (A) Smoothness maps

Unreal Engine's Megascans library uses (R) Ambient Occlusion, (G) Roughness, and (B) Displacement maps
